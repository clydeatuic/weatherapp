## Github Collaboration

1. Clone

```
$ git clone github-repo
$ cd github-repo
```

2. Modify Codes

```
$ git status
$ git config user.name "github-name-here"
$ git config user.email "email-address-here"

```

3. Pull

```
$ git pull

```

4. Push

```
$ git add .
$ git commit -m "commit msg here"
$ git push
```
